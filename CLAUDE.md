# İzin Talep Sistemi

## Proje

Bu proje Node.js ile geliştirilen bir web uygulamasıdır.

Amaç:

Çalışanların izin talebi oluşturabildiği, yöneticilerin bu talepleri onaylayıp reddedebildiği ve adminin sistemi yönetebildiği profesyonel bir web uygulaması geliştirmek.

## Teknolojiler

- Node.js
- Express.js
- MySQL
- HTML
- CSS
- JavaScript
- PWA

## Mimari

- MVC
- Controller
- Service
- Repository

## Roller

- Admin
- Yönetici
- Personel
